Информация о проекте:

Приложение, которое ускорит и автоматизирует процесс обучения нейронной сети

Функции:

  1. Сбор необходимых данных из интернета или готового источника.
  2. Автоматическое обучение по заданному алгоритму.
  3. Тестирование и создание отчета о результатах обучения

В процессе будет разработано:

  • тестовая нейронная сеть;
  • алгоритм поиска и выборки информации из источников;
  • алгоритм обучения;
  • алгоритм тестирования, результатом которого будет отчет в графическом виде;
  • графический интерфейс приложения;
  • сайт с информацией о проделанной работе.
  • алгоритм обучения;

Информация будет дополняться по мере разработки.
